# Food-Delivery-Analysis# Food Delivery Data Analysis

## Objective
Analyze food delivery data to extract insights on pricing, ratings, and customer preferences.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI

## Key Insights
- Restaurants are concentrated in major cities
- Popular cuisines dominate across regions
- Pricing does not always correlate with ratings
